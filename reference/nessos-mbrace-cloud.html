Brace

Cloud

Contains the methods (combinators) to express the primitive computations directly supported by the MBrace runtime.

Static members

Static memberDescription
Choice(computations)
Signature: (computations:seq<Cloud<'T option>>) -> Cloud<'T option>
Type parameters: 'T

Returns a cloud computation that will execute the given computations possibly in parallel and will return when any of the supplied computations have returned a successful value or if all of them fail to succeed. If a computation succeeds the rest of them are canceled. The success of a computation is encoded as an option type. This operator may create distribution.
GetProcessId()
Signature: unit -> Cloud<ProcessId>

Returns the ProcessId of the current process.
GetTaskId()
Signature: unit -> Cloud<string>

Returns the taskId of the current executing context.
GetWorkerCount()
Signature: unit -> Cloud<int>

Returns the number of worker nodes in the current runtime. This operator may create distribution.
Log(msg)
Signature: msg:string -> Cloud<unit>

Writes a string to the user logs.
Logf(fmt)
Signature: fmt:StringFormat<'T,Cloud<unit>> -> 'T
Type parameters: 'T

Writes a string to the user logs using the specified format.
OfAsync(asyncComputation)
Signature: asyncComputation:Async<'T> -> Cloud<'T>
Type parameters: 'T

Converts an asynchronous computation to a cloud computation.
Parallel(computations)
Signature: computations:seq<Cloud<'T>> -> Cloud<'T []>
Type parameters: 'T

Returns a cloud computation that will execute the given computations possibly in parallel and returns the array of their results. This operator may create distribution. If any exceptions are thrown all the results will be aggregated in an exception.
ToLocal(cloudComputation)
Signature: cloudComputation:Cloud<'T> -> Cloud<'T>
Type parameters: 'T

Converts a cloud computation to a computation that will be executed locally (on the same node).
Trace(cloudComputation)
Signature: cloudComputation:Cloud<'T> -> Cloud<'T>
Type parameters: 'T

Wraps a cloud computation in a computation that will return the same result but will also write trace information in the user logs.
